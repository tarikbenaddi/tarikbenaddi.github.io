### Research interests

* Optical communications for satellites\
<sup> R&D and a various system engineering activities on optical links for satellites communications: system architecture and analysis, physical layer design (digital signal processing, digital communication, modem, FEC...)</sup>


* Coding theory\
<sup> LDPC, turbo codes, iterative decoding, code design, EXIT/GEXIT charts, erasure codes, rateless codes</sup>

* Cognitive radio\
<sup> superposition, dirty paper coding, precoding, trellis shaping</sup>

* Digital communications\
<sup> linear (de)modualtion, continuous phase modulation, receiver complexity, equalization, synchronization, multiple access methods</sup>

* Machine learning applied to communications\
<sup> error correction code design, modulation</sup>
