Research interests

  • Optical communications for satellites
    R&D and a various system engineering activities on optical links for satellites communications: system architecture and analysis, physical layer design (digital signal processing, digital communication, modem, FEC…)

  • Coding theory
    LDPC, turbo codes, iterative decoding, code design, EXIT/GEXIT charts, erasure codes, rateless codes

  • Cognitive radio
    superposition, dirty paper coding, precoding, trellis shaping

  • Digital communications
    linear (de)modualtion, continuous phase modulation, receiver complexity, equalization, synchronization, multiple access methods

  • Machine learning applied to communications
    error correction code design, modulation
